# УБИ.215 — Угроза несанкционированного доступа к системе при помощи сторонних сервисов

## Описание

Угроза заключается в возможности осуществления нарушителем деструктивного программного воздействия на дискредитируемую систему с использованием сторонних легитимных сервисов (социальных сетей, мессенджеров, репозиториев кода и т.п.), используемых в качестве посредника.
Реализация данной угрозы возможна если дискредитируемая система уже скомпрометирована.

## Источник угрозы

Внешний нарушитель со средним потенциалом

## Объект воздействия

Программное обеспечение (программы)

## Нарушаемые свойства безопасности

| Свойство | Нарушается |
|----------|-----------|
| Конфиденциальность | Да |
| Целостность | Да |
| Доступность | Да |
