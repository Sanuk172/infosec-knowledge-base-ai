# AI-ассистент

Интеллектуальный помощник по базе угроз информационной безопасности ФСТЭК.
Задайте вопрос — ассистент найдёт ответ среди 227 угроз БДУ и объяснит его.

<AiChat />

## Как работает ассистент

Ассистент использует архитектуру **RAG** (Retrieval-Augmented Generation):

1. Ваш вопрос анализируется и сравнивается с базой из 227 угроз ФСТЭК
2. Находятся наиболее релевантные фрагменты
3. Языковая модель **Gemini 2.5 Flash** формирует ответ только на основе найденных данных
4. Ответ возвращается вместе с указанием источников (номера УБИ)

## Примеры вопросов

- *Какие угрозы связаны с BIOS?*
- *Что такое угроза деавторизации клиента беспроводной сети?*
- *Угрозы от внешнего нарушителя с низким потенциалом*
- *Какие угрозы нарушают конфиденциальность данных?*
