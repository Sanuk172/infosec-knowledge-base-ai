## Зачем нужен этот проект

Специалистам по информационной безопасности приходится работать с большим объёмом
регламентов, стандартов и практических инструкций. Ручной поиск по документам занимает
много времени и приводит к ошибкам. Данный портал решает эту задачу: все материалы
хранятся в единой структурированной базе, а встроенный AI-ассистент позволяет задать
вопрос на естественном языке и мгновенно получить ответ со ссылкой на источник.

## Как работает система целиком

1. **Эксперт** добавляет статью в Markdown-файл и отправляет изменение в Git.
2. **CI/CD** (GitHub Actions) автоматически собирает и публикует обновлённый сайт.
3. **Индексатор** обрабатывает новые страницы и обновляет векторный индекс для поиска.
4. **Пользователь** вводит вопрос — ассистент находит нужные фрагменты и формирует ответ.
5. **Модуль автодополнения** (C#) предлагает варианты поиска уже в процессе ввода.

| AI-ассистент | RAG на основе эмбеддингов |
| Модуль автодополнения | C# (.NET 10), бинарный поиск |
