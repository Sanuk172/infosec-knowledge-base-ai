# Раздел 2 — Практика и инструкции

Этот раздел содержит прикладные материалы: готовые процедуры, инструкции
и чеклисты, которые специалист может применить непосредственно в работе.

## Зачем нужен этот раздел

Теоретические знания из Раздела 1 необходимо воплощать в конкретные действия.
Раздел 2 отвечает на вопрос "как именно это сделать" и служит оперативным
справочником для сотрудников, работающих с инцидентами, аудитами и процедурами ИБ.

## Что будет размещено

- **Пошаговые инструкции** — действия при реагировании на инциденты, настройке средств
  защиты, проведении аудита.
- **Операционные чеклисты** — структурированные списки проверок для типовых задач ИБ.
- **Типовые ошибки** — разбор распространённых нарушений и способы их устранения.
- **Рабочие процедуры** — описание регулярных процессов: резервное копирование,
  управление доступом, мониторинг событий.

## Связь с AI-ассистентом

Статьи этого раздела написаны в формате, удобном для извлечения конкретных шагов.
Когда пользователь задаёт вопрос "как выполнить X", ассистент находит нужную
инструкцию и воспроизводит её шаги с указанием источника.

## Требования к актуальности

Практические материалы устаревают при изменении процессов или инструментария.
Каждая статья этого раздела должна содержать дату последнего пересмотра, чтобы
ассистент мог предупреждать пользователя о возможно устаревших данных.
