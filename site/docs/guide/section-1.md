# Раздел 1 — Теория и стандарты

Этот раздел является теоретической основой всей базы знаний.
Он закрывает потребность в вводном материале для тех, кто начинает работать
с темой информационной безопасности.

## Зачем нужен этот раздел

Без единого понятийного базиса специалисты в команде трактуют термины по-разному,
а AI-ассистент не может дать точный ответ без корректных определений в индексе.
Раздел 1 создаёт этот общий фундамент.

## Что будет размещено

- **Ключевые определения** — термины и понятия ИБ, стандартизированные для использования
  внутри проекта.
- **Обзор стандартов** — ISO/IEC 27001, NIST Cybersecurity Framework, ГОСТ Р 57580 и другие
  нормативные документы с кратким изложением требований.
- **Модели угроз** — STRIDE, PASTA и другие методологии анализа рисков.
- **Вводные рекомендации** — с чего начать выстраивание системы ИБ в организации.

## Связь с AI-ассистентом

Статьи этого раздела формируют базовый слой знаний, который ассистент использует
при ответе на общие вопросы. Чем точнее и однозначнее написаны определения,
тем выше качество ответов. Именно поэтому каждая статья имеет фиксированный формат:
**цель → контекст → материал → вывод**.

## Текущее состояние

Раздел подготовлен как структурная оболочка. Предметный контент по ИБ
добавляется командой проекта поэтапно согласно плану наполнения.
